**InsideAirbnb Data Dashboard**

This project was developed for the Human-Computer Interface (Interface Pessoa-Máquina) course.

The goal was to design and implement a data visualization platform using InsideAirbnb datasets. The interface was specifically built to help researchers, city officials, and activists analyze the impact of short-term rentals on local housing markets in cities like Lisbon and Porto.

*Project Phases*

Phase 1 (Prototyping): Created a high-fidelity mockup in Figma focusing on usability guidelines and specific user personas (Researcher, City Councilor, and Activist).

Phase 2 (Implementation): Developed the functional web interface using Vue.js, connecting to a mockup API to display listings, price trends, and occupancy maps.

*Key Features*

1. Interactive Dashboards: Time-series analysis and complex data filters.

2. Geographic Mapping: Visualization of listings across different city zones.

3. User-Centric Design: Tailored views for different technical skill levels, from raw data exports to simplified social media graphics.

How to run: npm run dev
